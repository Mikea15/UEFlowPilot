## FlowPilot Editor Overview

The editor is comprised of four main views. 
 <Image src={new_asset_view} alt="new_asset_view" />

- **Main Data**: Contains data shared from FlowPilot Data Asset.
- **Tree View**: Shows the overview of FlowPilot Task hierarchy.
- **Task Details**: Shows the selected Task -from the TreeView- details
- **Task Palette**: Displays the list of available tasks you can add to your FlowPilot Asset.
